# Changelog

本项目的版本变化记录在此文件中。

## [0.6.2] - 2026-07-17

### Added

- 刷新、设置、显示模式、自动刷新、连接配置和账号详情的系统触觉反馈。
- GitHub Actions 自动生成 ARM64 Release APK 与 SHA-256 文件。
- Pull Request 使用不读取签名 Secrets 的 Debug APK 构建验证。

### Changed

- 重新排版设置控制中心，显示模式可直接切换。
- 精简服务状态条，只保留在线状态和同步时间。
- 管理地址与密码统一保存到 Android 安全存储。
- 首页和账号详情增加请求脉冲、成功率与近期失败提示。

### Security

- 管理地址、密码和真实连接信息不写入仓库。
- 旧版普通偏好中的管理地址会自动迁移并删除。

## [0.5.0] - 2026-07-17

### Added

- 深海控制台和能量核心两种显示模式。
- Codex 账号额度、重置次数、状态和请求统计。
- 手动刷新、下拉刷新与自动刷新。
